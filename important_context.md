# Porcupine Important Context

이 파일은 포큐파인이 매 질문마다 참고하는 고정 참고사항이다.
오래된 대화 요약보다 우선해서 참고해야 한다.

## 현재 시점

- 현재 기준 연도는 2026년이다.
- 사용자가 "올해", "지금", "오늘"을 물으면 실행 환경에서 전달되는 현재 날짜/시간을 우선한다.
- 오래된 대화나 기억에 2023년 같은 과거 연도가 남아 있으면 현재 시점 판단에는 사용하지 않는다.

## 사용자와 가족

- 사용자는 1976년생이다.
- 사용자는 대전에서 어머니와 두 딸 소율, 소민과 함께 산다.
- 소율은 2015년생이다.
- 소민은 2018년생이다.
- 우리가족의 차는 볼보와 BMW이다. 
- 소율이와 소민이의 학교는 노은 초등학교이다. 
- 아빠의 회사는 인터그래비티이다. 
- 할머니는 성남에 사신다. 
- 외할아버지, 외할머니는 포항에 사신다. 
- 소율이와 소민이는 소설을 쓰고 있다. 
- 아빠의 전화번호는 010-2017-0184이야. 
- 엄마의 전화번호는 010-2695-1468이야.
- 우리집 주소는 대전 유성구 노은서로 124 노은 카운티스 109동 201호야. 

## 가족 취향

- 소율이가 좋아하는 음식: 민어조기. 치킨.
- 소민이가 좋아하는 음식: 딸기.
- 소율이가 좋아하는 책: 워리어즈.
- 소민이가 좋아하는 책: 무민.
